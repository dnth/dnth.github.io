Introduction

Microbiologist count cells manually. This process can be tedious and prone to mistakes. Why not automate the counting using a trained deep learning model? This blog post showcases MicroSense, a proof of concept product that can automate the counting of microalgae cells or any cells for that matter using deep learning and computer vision.

Training

Figure illustrates the raw detection of cells from microscope image. The model is a RetinaNet with a ResNet50 backbone trained using IceVision.

Hosting
